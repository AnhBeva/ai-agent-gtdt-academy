# AI Agent: Bounded Autonomy Academy

Static Apple-style training webpage for the Vietnamese AI Agent Bounded Autonomy academy.

## Source Content

- `Giao_trinh_AI_Agent_Bounded_Autonomy.md`

## Webpage

The generated static site lives in `webpage/` and is deployed to GitHub Pages by GitHub Actions.

To rebuild locally:

```bash
python3 webpage/build_webpage.py
```
